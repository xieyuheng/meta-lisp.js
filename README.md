# meta-lisp.js

A lisp as meta language to implement other languages.

## Usages

### Command line tool

Install it by the following command:

```sh
npm install -g @xieyuheng/meta-lisp.js
```

The command-line program is called `meta-lisp.js`.

## Development

```sh
npm install
npm run build
npm run test
```
