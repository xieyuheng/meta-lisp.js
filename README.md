# lattice-lisp.js

A lisp with structural type system.

## Usages

### Command line tool

Install it by the following command:

```sh
npm install -g @xieyuheng/lattice-lisp.js
```

The command-line program is called `lattice-lisp.js`.

## Development

```sh
npm install
npm run build
npm run test
```
